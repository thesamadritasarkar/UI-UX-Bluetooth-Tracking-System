# Bluetooth Tracker App – UI/UX Design

## Project Overview
This is a **UI/UX design project** for a Bluetooth tracker app, created in **Figma**. The app helps users locate lost items quickly and efficiently using Bluetooth connectivity and a simple, intuitive interface.  

## Problem Statement
Losing personal items like keys, wallets, or gadgets is common. Existing solutions can be clunky or confusing. The goal of this project is to **design a clean, user-friendly interface** that simplifies tracking and locating items.  

## Solution
- Real-time tracking of connected devices  
- Easy-to-understand interface for all age groups  
- Smooth navigation and interactive prototypes  
- Focus on minimalism, clarity, and accessibility

## Tools Used
- **Figma** – For UI/UX design and prototyping  
- **FigJam** – For flow diagrams and brainstorming  
- **Adobe Illustrator** (optional) – For custom icons  

## Design Process
1. **Research & User Flow** – Understanding user needs and defining app flows.  
2. **Wireframing** – Creating low-fidelity layouts for key screens.  
3. **High-Fidelity Design** – Designing polished screens with Figma.  
4. **Prototyping** – Linking screens to create interactive flows.  
5. **Testing & Iteration** – Refining designs based on usability feedback.
